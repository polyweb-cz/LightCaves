# Story 4.2: Main Menu

## User Story

**As a** hráč
**I want to** vidět hlavní menu s možností začít hru
**So that** mám přehledný vstupní bod do aplikace

## Popis

Vytvořit systém hlavního menu:
- Zobrazení menu na startu aplikace
- Tlačítka: Start, Nastavení, O hře, Konec
- Animace při přechodu z menu do hry
- Návrat do menu z hry
- Logo a verze aplikace

## Acceptance Criteria

- [ ] Třída MainMenu
- [ ] showMenu() - zobrazí menu
- [ ] hideMenu() - skryje menu
- [ ] onButtonClick(button) event handler
- [ ] Menu buttons: Start, Settings, About, Quit
- [ ] Keyboard navigation (Tab, Enter)
- [ ] Unit testy: 10 testů
- [ ] npm test projde

## Odhad

- 0.5 dne (menu renderer + testy)
