# Story 4.3: Level Selection Menu

## User Story

**As a** hráč
**I want to** vybrat si level ze seznamu
**So that** mohu hrát různé levely v libovolném pořadí

## Popis

Vytvořit menu pro výběr levelů:
- Zobrazení seznamu dostupných levelů
- Indikátor: zamčený, odemčený, dokončený
- Zobrazení best stats pro každý level
- Filtrování dle obtížnosti
- Vyhledávání levelů

## Acceptance Criteria

- [ ] Třída LevelSelector
- [ ] showLevelList() - zobrazí seznam levelů
- [ ] selectLevel(levelId) - vybere level
- [ ] getLevelStats(levelId) - vrátí best stats
- [ ] Unit testy: 10 testů
- [ ] npm test projde
