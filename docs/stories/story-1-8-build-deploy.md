# Story 1.8: Build & Deploy

## User Story

**As a** vývojář
**I want to** mít hotový build pipeline a GitHub Pages setup
**So that** mohu snadno nasadit na web

## Acceptance Criteria

- [ ] npm run build vytvoří /dist folder
- [ ] Všechny assety jsou minifikovány
- [ ] Build je < 10MB
- [ ] .gitignore ignoruje /dist a /node_modules
- [ ] GitHub Actions (CI/CD) konfigurován
- [ ] Deploy instrukce v README.md
- [ ] Žádné build warningy
- [ ] Production build funguje

## Odhad

- 0.5 dne (setup GitHub Actions, README)
