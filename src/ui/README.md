# UI Module

User interface components and interactions.

## Files

- **ui.js** - Main UI manager (menus, modals, HUD)
- **palette.js** - Mirror selection palette

## Responsibilities

- Render menus (main, level select, settings)
- Display game HUD (level name, progress)
- Handle mirror palette (selection, placement preview)
- Show victory/completion modals
- Manage keyboard/mouse input UI feedback

## Key Concepts

- UI is separate from game logic
- Event-driven architecture
- CSS-based styling (see /src/styles.css)
